Opcodes
-------
0x00  NOP       - No operation
0x01  POKE      - POKE [addr] <- value
0x02  MOVE      - MOVE [src] -> [dest]
0x03  NOT       - NOT [addr]
0x04  NAND      - NAND [a] [b] -> [dest]
0x05  JMP       - Jump to [addr]
0x06  JZ        - Jump to [addr] if [cond]==0
0x07  RUN       - Push return address; jump to [addr]
0x08  HALT      - Stop execution / return if stack
0x09  AND       - AND [a] [b] -> [dest]
0x0A  OR        - OR [a] [b] -> [dest]
0x0B  XOR       - XOR [a] [b] -> [dest]
0x0C  INC       - Increment [addr]
0x0D  DEC       - Decrement [addr]
0x0E  CMP       - Compare [a] [b] -> [dest] (1 if equal, else 0)
0x0F  COMMENT   - Skip [len] bytes

I/O Opcodes
-------
0x10  PUTC      - Output character from [addr]
0x11  PUTN      - Output decimal number from [addr]
0x12  GETC      - Input character -> [addr]
0x13  RET       - Return from RUN / RUN16

Arithmetic & Shifts
-------
0x14  ADD       - [a] + [b] -> [dest]; sets overflow_flag
0x15  SUB       - [a] - [b] -> [dest]; sets overflow_flag
0x16  MUL       - [a] * [b] -> [dest]; sets overflow_flag
0x17  DIV       - [a] / [b] -> [dest]
0x18  SHL       - [a] << ([b] & 0x07) -> [dest]
0x19  SHR       - [a] >> ([b] & 0x07) -> [dest]

16-bit Addressing Opcodes
-------
0x1A  POKE16    - POKE16 [addr16] <- value
0x1B  MOVE16    - MOVE16 [src16] -> [dest16]
0x1C  JMP16     - Jump to [addr16]
0x1D  JZ16      - Jump to [addr16] if [cond]==0
0x1E  RUN16     - Push return; jump to [addr16]

Memory & Stack
-------
Memory: 64K unified memory (0x0000â€“0xFFFF)
Stack:  64-level call stack storing 16-bit return addresses
Overflow Flag: Set when arithmetic operations exceed 8-bit limits
Instruction Limit: 1,000,000 instructions max (prevents infinite loops)

Execution Notes
-------
- Self-modifying code is allowed since memory is unified.
- Instruction operands are bounds-checked to avoid memory faults.
- Debug and trace modes can help trace instruction execution.
- Comments are skipped by loader; both ';' and '#' are supported.
